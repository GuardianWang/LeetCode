## Tricks

- `if n & 1`: odd

### O(logN) power of N
```python
def pow(a, n):
  remain = 1
  power = 1
  while power <= n:
    # a is input_a ^ power
    if power & n:
      remain *= a
    a *= a
    power >>= 1
  return a
```
